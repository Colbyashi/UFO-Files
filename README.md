# UFO-Files
This is just an easy place to download all of the U.S. UFO Files all at once. Since no "download all" button exists on war.gov/UFO, I've downloaded everything manually so you don't have to.


## Source:
https://www.war.gov/UFO/


## Discrepencies:
Currently, the website displays a total of 161 documents and files. You will notice in the zip file there are only 157. The reasons being:

1.) There are 3 duplicate documents on the website.

2.) At the time of downloading all of these documents, there was also one document that threw a "Not Found" error when trying to download (https://www.war.gov/UFO/#65_HS1-834228961_62-HQ-83894_Serial_153).

Some photos were uploaded directly to the website as PNGs. But a large number of photos were placed into PDFs for some reason. I've taken the time to extract those photos and save them as PNGs for easier viewing. These re-saved PNGs have the same names as their corresponding PDFs.


## Maintenance:
As more files get released, I plan on adding them to this repository so people have an easy wasy to access and view the documents. I also plan on extracting all the text provided with each document from the website so each document can be viewed with its full, original context.
